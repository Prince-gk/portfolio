My Projects

Project 1: Personal Blog Platform

This project is a comprehensive blogging platform that I built using the MERN stack (MongoDB, Express, React, Node.js). The platform allows users to create accounts, write, and manage blog posts, and engage with other users through comments and likes. The back-end includes secure user authentication, and the entire application is designed to be highly scalable.

Key Features:

  • Custom content management system (CMS) for easy post creation
  • Responsive design with a focus on clean typography
  • User authentication using JWT and OAuth (Google)
  • Real-time comment and like functionality using WebSockets
This project highlights my ability to create full-stack applications with a focus on delivering engaging user experiences.

Project 2: E-commerce Website

This e-commerce platform was developed for a local business, providing them with a complete online shopping experience. Built using React for the front-end and Node.js for the back-end, the site allows customers to browse products, add items to their cart, and securely check out using Stripe for payment processing.

Key Features:

  • Dynamic product catalog with filtering by category, price, and availability
  • Shopping cart with persistent data stored in local storage
  • Admin panel for product management (CRUD operations)
  • Secure checkout process with Stripe integration
  • Real-time inventory management to track stock levels
The challenge in this project was to ensure the platform remained performant, even as the product catalog grew, which required careful optimization of the back-end database queries and caching.

Project 3: Task Manager App

This project was built to help users manage their personal tasks and projects efficiently. Using React for the front-end and Firebase for the back-end, the app allows users to create tasks, set deadlines, and prioritize their work with a simple drag-and-drop interface.

Key Features:

  • Real-time updates and synchronization using Firebase Firestore
  • Task categorization and deadline reminders
  • Drag-and-drop functionality for task prioritization
  • Responsive design optimized for both desktop and mobile use
  • Custom notifications for upcoming tasks and deadlines
This project taught me the importance of optimizing for performance, especially when dealing with real-time updates and user data synchronization across devices.

Project 5: Weather Forecasting App

This is a weather forecasting application built using React and the OpenWeatherMap API. The app fetches real-time weather data for any location in the world and displays it in a user-friendly interface, including temperature, humidity, wind speed, and more.

Key Features:

  • Real-time weather data fetched from OpenWeatherMap API
  • Interactive search functionality with auto-suggestions
  • Geolocation-based weather data for current location
  • 5-day weather forecast displayed graphically
This project provided valuable experience in API integration, data handling, and UI/UX design for data-heavy applications.
